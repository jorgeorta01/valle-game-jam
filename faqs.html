Valle Game Jam 2024

What is a Game Jam?

A game jam is an event where participants create a game from scratch, typically in teams comprising programmers, writers, artists, and game designers. The event includes a theme, restrictions, and a time limit, announced one week before.

Follow @vallegamedevs on Instagram for updates.
