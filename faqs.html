What is a Game Jam?

A game jam is blah blah blah, usually following a theme and time limit. However, the Valle Game Jam is 48 hours and follows a game format instead. The game format is our focus. This includes a theme, restriction and a time limit. The game format will be revealed one week in advance before the event begins.
